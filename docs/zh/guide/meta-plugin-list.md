# 已有的元插件 (Meta-Plugins)

元插件 (MetaPlugin) 是 Xinbot 的核心组件，负责处理特定服务器的连接、登录及排队逻辑。本页列出了目前社区提供的元插件。

如果你开发了新的元插件，欢迎通过 [提交 Issue](https://github.com/huangdihd/xinbot_documention/issues/new?template=meta_plugin_submission.yml) 来申请添加到此列表。
