# Current Meta-Plugins

Meta-Plugins are core components of Xinbot, responsible for handling connection, login, and queuing logic for specific servers. This page lists the Meta-Plugins currently provided by the community.

If you have developed a new Meta-Plugin, feel free to [Submit an Issue](https://github.com/huangdihd/xinbot_documention/issues/new?template=meta_plugin_submission.yml) to request its addition to this list.
